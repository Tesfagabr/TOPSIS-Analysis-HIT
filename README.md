AHP-TOPSIS Road Ranking & Sensitivity Analysis
This repository contains one MS Excel file for data input and two Python scripts for multi-criteria decision analysis using AHP-TOPSIS methodology, designed for ranking road alternatives based on multiple criteria, followed by comprehensive sensitivity analysis.

Part 2:  Input Data Format
File Structure
The code directly inputs road characteristics data from an Excel file named TOPSIS Input Data.xlsx. Thus, users are expected to input the data into the two specific Excel worksheets:
Sheet 1: Global Weight (AHP Computed weight)
Requirements:
•	Column headers must be exactly "Criteria" and "Global Weight" (case-sensitive)
•	Criteria names must match those in the Alternatives sheet
•	Sum of weights should be 1.0 (code auto-normalizes if needed)
•	Include a "Total" row at the bottom (optional but recommended for visual check)
•	✓ No merged cells in the data area
•	✓ Column A contains unique road names
•	Excel file is not open in another program while running the script
Sheet 2: Alternatives
Layout Rules:
•	Row 2: Criteria Benefit or Non-benefit indicators
•	Row 3: Criterion names (must always match names in Global Weight sheet)
•	Row 4 to End: Road names in column A, numeric values in columns B through K (or up to last criterion)
Important Notes:
•	Don’t leave empty cells in the data matrix to avoid analysis error
•	All criteria data must be converted to numeric values
•	Column order must be consistent across sheets
Part 3:  How to Run
Step 1: Prepare Input File
Place the TOPSIS Input Data.xlsx file in the same directory as the Python scripts.
Step 2: Run TOPSIS Calculation
Run the Python code "TOPSIS Ranking.py". This performs the initial TOPSIS ranking and generates TOPSIS_Results.xlsx. file in the same directory.
Step 3: Run Sensitivity Analysis
Run the Python code "Final Sensitivity Analysis.py". This reads the normalized matrix from the generated TOPSIS_Results.xlsx file and performs a comprehensive sensitivity analysis. The analysis result sheet for this computation will be created inside TOPSIS_Results.xlsx.

Part 4:  Output Files
1. TOPSIS_Results.xlsx (Generated by TOPSIS Ranking.py)
After running the TOPSIS Ranking.py, a TOPSIS_Results.xlsx file will be created, that contains the following 6 worksheets:

Sheet Name	Description
AHP Weights	Normalized criteria weights from input
Alternatives Data	Raw input data for verification
Normalized Matrix	Vector-normalized decision matrix
Weighted Matrix	Normalized × Weights
Ideal Solutions	Ideal best and worst values per criterion
TOPSIS Results	Final scores, S+, S-, and rankings (sorted by rank)
2. Updated TOPSIS_Results.xlsx (After Sensitivity Analysis)
The sensitivity script adds a new worksheet to the existing TOPSIS_Results.xlsx file:

Sheet Name		Content
Rank Sensitivity		Comprehensive sensitivity results, including:
•	Base TOPSIS results 
•	Criteria impact metrics
•	Road stability metrics
•	Full sensitivity results for all criteria
		
		

3. Visual Outputs (Displayed on Screen)
The sensitivity analysis will automatically generate 6 plots:
1.	Rank Frequency Heatmap: How often each road gets each rank
2.	Criteria Impact Bar Chart: Which criteria cause the most rank changes
3.	Average Rank with Variability: Mean rank plus or minus (±) standard deviation per road
4.	Rank Distribution Heatmap: Frequency distribution of ranks
5.	Rank Stability Lines: Line plots per criterion (weight change vs. rank)
6.	Enhanced Impact Bar Chart: Normalized impact scores (0-100)
